# Calendar Task Planner

A clean monthly calendar planner for Obsidian to manage tasks day by day.

## Features

- Full monthly calendar view (week starts Monday)
- Tasks with checkboxes directly on day cards
- Add, edit, delete tasks in a modal window
- Monthly statistics (total / done / percentage)
- Export day to a note (2026-06-01.md)
- Add day to Obsidian Canvas
- All data stored locally in `Calendar Task Planner.json`
- Works fully offline

## Installation

### Manual
1. Download `main.js`, `styles.css`, `manifest.json` from the latest release
2. Copy to `.obsidian/plugins/calendar-task-planner/`
3. Enable in Settings → Community plugins

### Community plugins
Search for **Calendar Task Planner** in Obsidian's community plugin browser.

## Data & Privacy

All data is stored locally in your vault as `Calendar Task Planner.json`.
No network requests are made.
